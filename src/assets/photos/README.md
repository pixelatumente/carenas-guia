# Fotos de lugares

Añade aquí las fotos reales de cada ficha.

## Convención recomendada

- Foto principal: `nombre-del-lugar.jpg`
- Galería: `nombre-del-lugar-1.jpg`, `nombre-del-lugar-2.jpg`, etc.

Ejemplo para la ficha `castillo-de-somet`:

```text
src/assets/photos/castillo-de-somet.jpg
src/assets/photos/castillo-de-somet-1.jpg
src/assets/photos/castillo-de-somet-2.jpg
```

Formatos admitidos: JPG, JPEG, PNG, WebP y AVIF.
